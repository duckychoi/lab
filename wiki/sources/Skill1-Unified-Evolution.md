# Skill1: Unified Evolution of Skill-Augmented Agents via RL

> [!insight] 핵심 인사이트
> 에이전트가 스킬을 수동으로 설계받지 않고 RL을 통해 스스로 스킬 라이브러리를 진화시키는 통합 프레임워크. [[SkillClaw]], [[Co-Evolving-LLM-Agents]]와 동일 방향 — "에이전트가 자신의 능력을 스스로 확장한다"는 패러다임이 주류로 부상 중.

## 도메인별 추출 (ai-news)

- **신뢰도**: HF 논문 업보트 58 (2026-05-09) — 학술 논문, arXiv 2605.06130
- **즉시 활용**: 아직 구현체 없음. 개념 이해 및 향후 오픈소스 구현 모니터링
- **6개월 영향력**: 스킬 자동 진화가 에이전트 프레임워크 표준이 되면 수동 스킬 설계 작업량 대폭 감소
- **대체 관계**: 수동 스킬 라이브러리 설계([[agent-skills]], [[awesome-agent-skills]]) 패러다임을 장기적으로 대체
- **허와 실**: RL 기반 진화는 계산 비용 높음. 실용화까지 시간 필요
- **액션**: [[SkillClaw]]와 함께 에이전트 스킬 진화 클러스터로 묶어 synthesis 작성 검토

> [!note] 배경 정보
> 에이전트 스킬 자동화 방향: 수동 설계 → RL 진화 → 자기증식형 스킬뱅크. Skill1은 이 진화의 중간 단계를 다룸.
