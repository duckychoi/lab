# VoxCPM2

> [!insight] 핵심 인사이트
> OpenBMB의 30개 언어 지원 TTS 모델. Apache 2.0 라이선스로 상업용 음성 합성 즉시 활용 가능. 1,198 likes로 음성 복제까지 통합.

## 핵심 인사이트

> [!insight] 왜 중요한가
> HF 트렌딩 11위 (66,600 다운로드, 1,198 likes), Apache 2.0 — [[VoxCPM]](기존 버전, ⭐14,227)의 후속. 30개 언어 지원과 음성 복제 결합이 핵심 강점. [[OmniVoice]](1M+ 다운로드)와 직접 경쟁하는 오픈소스 TTS 모델.

> [!action] 당장 할 것
> HuggingFace에서 직접 다운로드 후 [[silero-vad]]와 결합해 TTS 파이프라인 테스트. 한국어 지원 품질 확인.

- **신뢰도**: HF 1,198 likes, 66.6K 다운로드, OpenBMB 공식, Apache 2.0 — high
- **즉시 활용**: YES — Apache 2.0 상업용 즉시 사용 가능. 30개 언어 한국어 포함
- **6개월 영향력**: 오픈소스 TTS 표준 모델 위치 확보 가능성. [[OmniVoice]] 대항마

- **허와 실**: 음성 품질은 직접 테스트 필요. 30개 언어 품질 편차 있을 수 있음
- **액션**: 즉시 설치+테스트
