# OccuBench

> [!insight] 핵심 인사이트
> Qwen 팀의 실세계 직업 전문 태스크 AI 에이전트 평가 벤치마크. 업보트 38. 언어 월드 모델 기반으로 현실적 업무(변호사, 의사, 엔지니어 등 직업군 태스크)에서 에이전트 수행 평가.

## 핵심 인사이트

**핵심 기여:**
- "직업별 실무 태스크"를 에이전트 평가 기준으로 사용
- Language World Model 기반 검증 가능한 환경
- Qwen 팀 → Alibaba 계열, 강력한 리소스 배경

> [!note] 배경 정보
> [[GameWorld]], [[ClawBench]]와 함께 에이전트 벤치마크 트리오. OccuBench의 차별점: "게임"이나 "일상 태스크" 아닌 **직업 전문성** 기준 평가.

> [!action] 당장 할 것
> 에이전트 성능 주장 시 OccuBench 점수 기준으로 신뢰도 평가하는 기준선 삼기
