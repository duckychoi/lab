# ClawGUI: A Unified Framework for GUI Agents

> [!insight] 핵심 인사이트
> GUI 에이전트의 학습·평가·배포를 단일 프레임워크로 통합. 앱/웹/데스크톱 가리지 않고 자동 태스크 수행 — 클릭, 입력, 스크롤 등 UI 조작을 LLM이 직접 제어.

## 핵심 인사이트

> [!action] 당장 할 것
> [[ClawBench]]와 함께 검토. GUI 에이전트 자동화 파이프라인 구성 시 레퍼런스 논문으로 활용.

> [!note] 배경 정보
> [[AI-에이전트-프레임워크]] 흐름의 연장선. GUI 레이어에서의 에이전트화는 RPA(Robotic Process Automation)를 LLM 기반으로 대체하는 방향.
