# baidu/ERNIE-Image

> [!insight] 핵심 인사이트
> Baidu의 8B Apache 2.0 텍스트-이미지 생성 모델 — diffusers 기반, fal.ai 추론 지원. ERNIE 생태계 이미지 생성 공개로 Baidu가 이미지 생성 오픈소스 경쟁에 공식 참전. FLUX, SDXL 기반 모델 생태계에 중국 대형 기업이 직접 투입.

## 도메인별 추출

**신뢰도**: ⭐⭐⭐ (HF 3,800 다운로드, Baidu 공식 레포 — 초기 공개)
**즉시 활용**: YES — diffusers 통합으로 즉시 사용 가능. `from diffusers import ...`
**6개월 영향력**: ERNIE 언어 모델과 이미지 생성의 멀티모달 통합 경로 주목
**대체 관계**: SDXL, FLUX 계열의 중국 자체 대안

> [!note] 배경 정보
> Baidu는 ERNIE(Enhanced Representation through kNowledge Integration) 브랜드로 언어모델 생태계 구축 중. 이미지 생성 모델 공개는 멀티모달 확장의 신호.

## 주요 기능

- 텍스트 프롬프트 → 고품질 이미지 생성
- 8B 파라미터, Apache 2.0 라이선스
- HuggingFace diffusers 완전 호환
- fal.ai 클라우드 추론 지원
