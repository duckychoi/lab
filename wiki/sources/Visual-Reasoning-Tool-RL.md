# Visual Reasoning via Tool-RL

> [!insight] 핵심 인사이트
> Amazon AGI Research가 제안한 도구 활용 강화학습(Tool-RL) 기반 시각 추론 향상 방법론. VLM이 외부 도구(계산기, 검색, 코드 실행)를 자율 선택·활용하도록 RL 훈련 — 순수 모델 파라미터 확장보다 도구 활용 능력 향상에 집중.

## 도메인별 추출

**즉시 활용**: 영상 분석 파이프라인에서 VLM + 도구 조합 시 참고 가능. 직접 적용보다 아이디어 차용 수준.

**6개월 영향력**: VLM이 도구를 자율 선택하는 패턴이 표준화되면 [[AI-에이전트-프레임워크]] 설계 방식이 바뀜 — "모델 자체 능력"보다 "도구 오케스트레이션 능력"이 핵심 경쟁력.

**허와 실**: 2 upvotes. 논문 단계. Amazon AGI 내부 연구. 상용화까지 거리 있음.

**대체 관계**: [[cognee]], [[에이전트-메모리-레이어]]와 상호보완 — 메모리는 장기 컨텍스트, Tool-RL은 실시간 추론 도구 활용.
