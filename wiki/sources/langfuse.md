# langfuse/langfuse

> [!insight] 핵심 인사이트
> LLM 앱의 메트릭·평가·프롬프트 관리를 통합한 오픈소스 관측성 플랫폼. 25K 스타로 LLMOps 영역에서 사실상 표준. 프로덕션 LLM 앱 운영 필수 인프라.

## 도메인별 추출

- **신뢰도**: 25K 스타, SaaS + 오픈소스 양립 — 기업 신뢰도 높음
- **즉시 활용**: 프롬프트 성능 추적, A/B 테스트, 비용 모니터링 즉시 적용 가능
- **6개월 영향력**: LLM 앱의 "디버깅·최적화 루프" 표준화. 품질 없는 LLM 앱 개발은 맹목적 시도
- **대체 관계**: LangSmith 오픈소스 대안. 자체 호스팅 가능 = 데이터 주권 확보
- **메모리/에이전트 연결**: [[cognee]]·[[claude-mem]]과 연동하면 에이전트 메모리 품질도 추적 가능
- **액션**: reat-voice / wiki 파이프라인에 프롬프트 추적 레이어 추가 검토
