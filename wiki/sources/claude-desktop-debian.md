# aaddrick/claude-desktop-debian

> [!insight] 핵심 인사이트
> Claude Desktop을 Debian/Ubuntu 리눅스에서 네이티브 실행 — Anthropic이 공식 지원하지 않는 Linux 환경에서 MCP 통합 데스크탑 환경을 제공. 리눅스 사용자가 MCP 에코시스템 전체를 접근할 수 있는 유일한 경로.

## 도메인별 추출

**즉시 활용**: YES (Linux 환경) — 설치 후 Claude Desktop + MCP 서버 전체 생태계 활용
**신뢰도**: ⭐⭐⭐ (커뮤니티 프로젝트, ⭐3,515, 비공식이므로 업데이트 지연 리스크)
**대체 관계**: CLI 전용 Claude Code 대비 GUI + MCP 통합 환경 제공

> [!warning] 주의
> 비공식 래퍼 — Anthropic의 내부 API 변경 시 동작 중단 가능성. 보안 감사 미완료 상태.

## 주요 기능

- Claude Desktop 공식 앱을 Debian 계열 Linux에 패키징
- MCP 서버 통합 지원 (chrome-devtools-mcp 등)
- AppImage/deb 패키지 형태로 배포
