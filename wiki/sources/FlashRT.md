# FlashRT: Red-Teaming for Long-context LLMs (arXiv:2604.28157)

**arXiv**: https://arxiv.org/abs/2604.28157  
**신뢰도**: ⭐⭐⭐

> [!insight] 핵심 인사이트
> 프롬프트 인젝션·지식 오염 공격의 연산/메모리 비용을 2~7배 가속, 2~4배 메모리 절감. 32K 토큰 기준 264GB→65GB GPU 메모리. LLM 안전 연구의 계산 병목을 해결 — 더 많은 취약점을 더 빠르게 발견 가능.

> [!warning] 주의 / 신뢰도 낮음
> 레드팀(공격) 효율화 도구는 방어 연구 목적이지만 악용 가능성도 있음. 오픈소스 공개 범위와 접근 제한 여부 확인 필요.
