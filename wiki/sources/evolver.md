# EvoMap/evolver

> [!insight] 핵심 인사이트
> 에이전트 행동을 유전자처럼 취급해 진화 알고리즘으로 자동 최적화 — "프롬프트 엔지니어링"을 수동 작업 아닌 진화 과정으로 대체. 최적 행동 패턴을 세대 반복으로 발견.

## 도메인별 추출

**즉시 활용**: 중간 — 실험적 프레임워크, 프로덕션 미검증
**신뢰도**: ⭐⭐⭐ (⭐5,147, Genome Evolution Protocol 개념 독창적)
**6개월 영향력**: 에이전트 자기최적화 패러다임의 실용화 신호 — [[AI-에이전트-프레임워크]] 진화 방향
**대체 관계**: 수동 프롬프트 엔지니어링, DSPy 자동 최적화 대비 진화 기반 접근

> [!warning] 주의
> "자기진화" 마케팅 언어 주의. 실제 벤치마크/재현성 검증 필요.

## 주요 기능

- Genome Evolution Protocol: 에이전트 행동을 유전자(genome)로 인코딩
- 세대 반복으로 행동 패턴 진화·최적화
- 다양한 태스크에서 최적 에이전트 행동 자동 탐색
