# Semi-DPO: Semi-Supervised DPO (arXiv:2604.24952)

**arXiv**: https://arxiv.org/abs/2604.24952  
**신뢰도**: ⭐⭐⭐

> [!insight] 핵심 인사이트
> 다차원 시각 선호(색상, 구도, 스타일 등)를 단순 이진 레이블(선호/비선호)로 압축할 때 발생하는 정보 손실·노이즈 문제를 반지도학습으로 해결. 추가 어노테이션 없이 기존 이진 레이블 데이터만으로 복잡한 인간 선호를 더 정확히 반영.

## 도메인별 추출 (ai-news 템플릿)

- **신뢰도**: ⭐⭐⭐ — arXiv 논문, 학술적 기여
- **즉시 활용**: 연구 레퍼런스 — DPO 기반 시각 모델 파인튜닝 시 레이블 노이즈 처리 방법론으로
- **6개월 영향력**: 이진 선호 레이블의 한계를 극복하는 정렬 기법 발전
- **대체 관계**: [[ViPO]](대규모 선호 데이터셋)와 상호보완 — 데이터 양 vs 레이블 품질 접근
- **액션**: 선호 데이터 품질 문제 겪을 시 semi-supervised 접근 검토
