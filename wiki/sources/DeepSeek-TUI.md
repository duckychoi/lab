> Rust 기반 TUI(Terminal User Interface)로 DeepSeek 모델에 접근하는 경량 코딩 에이전트. [[DeepSeek-V4-Pro]](535K DL)의 인기를 TUI 형태로 접근하려는 수요 반영. 2.8K→5K 급성장은 "터미널에서 DeepSeek" 니즈가 상당함을 보여줌.

> Rust 기반 → 네이티브 성능, 의존성 최소화. Claude Code·Gemini CLI와 같은 터미널 AI 에이전트 트렌드의 DeepSeek 버전.
