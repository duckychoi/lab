# CloakHQ/CloakBrowser

> [!insight] 핵심 인사이트
> AI 에이전트의 웹 자동화 최대 장벽은 봇 탐지(Cloudflare, DataDome 등). CloakBrowser는 Playwright 드롭인 교체만으로 30/30 탐지 테스트를 통과시키는 스텔스 Chromium. 에이전트 웹 브라우징의 실용성을 크게 높이는 인프라 레이어.

## 도메인별 추출 (ai-news)

- **신뢰도**: GitHub ⭐3,551 (+526 오늘) — 신생 레포, 급상승 중. 검증 중간 수준
- **즉시 활용**: YES (단, 윤리/법적 검토 필요) — Playwright 코드 `playwright` → `cloakbrowser` 교체만으로 적용
- **6개월 영향력**: AI 에이전트 웹 자동화 표준 인프라로 자리잡을 가능성. [[chrome-devtools-mcp]]와 보완 관계
- **대체 관계**: 기존 puppeteer-stealth, playwright-stealth 플러그인 대체
- **허와 실**: 봇 탐지 우회 능력은 서비스 약관 위반 위험. 리서치/스크래핑 목적 명확히 해야 함
- **액션**: 에이전트 웹 리서치 파이프라인에 통합 검토

> [!warning] 주의 / 신뢰도 낮음
> 봇 탐지 우회 도구는 서비스 이용약관(ToS) 위반 가능성이 있음. 적법한 용도(자체 서비스 테스트, 공개 데이터 수집 등)에만 사용할 것. "30/30 탐지 통과" 클레임은 독립 검증 필요.
