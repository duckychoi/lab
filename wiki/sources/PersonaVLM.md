# PersonaVLM: Long-Term Personalized Multimodal LLMs

> [!insight] 핵심 인사이트
> 사용자 페르소나를 장기 기억하며 개인화된 멀티모달 응답을 생성하는 VLM 프레임워크 — 대화 일관성 대폭 개선. [[에이전트-메모리-레이어]]와 [[cognee]]가 텍스트 메모리 레이어를 해결하듯, PersonaVLM은 시각 정보 포함 개인화 메모리 레이어를 탐구.

## 도메인별 추출

**신뢰도**: ⭐⭐⭐ (arXiv 2604.13074, upvote 19, 미검토 논문)
**즉시 활용**: NO — 아직 연구 단계, 코드 공개 여부 확인 필요
**6개월 영향력**: AI 비서 장기 개인화 방향의 핵심 레퍼런스 될 가능성
**대체 관계**: 단순 RAG + 프로필 정보 주입 방식의 고도화

## 주요 기능

- 사용자별 페르소나(선호·습관·스타일) 장기 저장
- 멀티모달(이미지+텍스트) 개인화 응답
- 대화 일관성 평가 프레임워크 포함

> [!note] 배경 정보
> 현재 AI 비서 대부분은 세션 간 개인 정보를 기억하지 못함. PersonaVLM은 이 문제를 멀티모달 레벨에서 해결하는 연구.
