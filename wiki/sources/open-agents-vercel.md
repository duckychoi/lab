# Vercel Open Agents

> [!insight] 핵심 인사이트
> Vercel 공개 클라우드 에이전트 빌딩 템플릿. ⭐2,820(당일 +915 — 첫날 급등). 프로덕션 수준 AI 에이전트를 Next.js + Vercel 배포 환경에서 빠르게 구축하는 오픈소스 출발점.

## 핵심 인사이트

**스펙:**
- GitHub ⭐2,820 (당일 +915, 신규)
- 개발사: Vercel Labs (공식)
- 기술 스택: Next.js, Vercel AI SDK, 서버리스

> [!note] 배경 정보
> [[hermes-agent]]·[[superpowers]]·[[multica]] 같은 복잡한 에이전트 프레임워크와 달리, Vercel의 접근은 "배포 우선" — 인프라 없이 에이전트 프로덕션 배포에 집중.

**차별점:**
- Vercel 생태계와 즉시 통합 (Edge Functions, KV, Blob)
- AI SDK 기반 표준화된 LLM 호출 인터페이스
- 서버리스 → 스케일링 자동화
