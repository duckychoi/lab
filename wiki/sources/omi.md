# BasedHardware/omi

> [!insight] 핵심 인사이트
> 화면 인식 + 음성 인식을 결합한 멀티모달 오픈소스 AI 비서. ⭐9,983(당일 +824). 하드웨어 웨어러블 디바이스와 연동 지원 — 모바일/데스크탑 소프트웨어에서 오픈소스 착용형 AI 기기까지 포괄하는 이례적 스택.

## 핵심 인사이트

**스펙:**
- GitHub ⭐9,983 (당일 +824)
- 핵심 기능: 화면 캡처 + 음성 입력 실시간 처리
- 하드웨어: 오픈소스 웨어러블 AI 기기(OMI 디바이스) 연동
- 라이선스: MIT

**기술 스택:**
- 멀티모달 입력: 화면 인식(비전) + 음성 인식(STT) 동시 처리
- 에이전트 루프: 입력 → 컨텍스트 파악 → 행동 제안/실행
- 하드웨어 레이어: 목걸이형 소형 디바이스(마이크 내장)

> [!note] 배경 정보
> [[BasedHardware]] 팀이 개발. 소프트웨어 에이전트와 피지컬 웨어러블을 동시에 오픈소스로 공개하는 전략은 드문 사례. AI Pin·Rabbit R1 같은 상용 기기 대비 오픈 생태계.

> [!action] 당장 할 것
> 소프트웨어 전용 모드(웨어러블 없이)로 먼저 테스트 — 화면+음성 멀티모달 캐치 능력 벤치마크

> [!question] 미해결 질문
> 실제 웨어러블 없이 PC/모바일 소프트웨어만으로 유용한가? 한국어 음성 인식 품질은?

## 도메인별 추출

- **즉시 활용**: 조건부 — 소프트웨어 모드는 즉시 가능, 웨어러블은 하드웨어 구매 필요
- **대체 관계**: AI Pin, Rabbit R1 대비 오픈소스 + 커스터마이징 자유도 높음
- **6개월 영향력**: 웨어러블 AI 기기 오픈소스 생태계 형성의 기점이 될 수 있음
- **허와 실**: 9,983⭐ 급상승이지만 실제 하드웨어 완성도는 검증 필요
