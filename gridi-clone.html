Solution

gridi는 사람처럼 서비스를 직접 탐방합니다

✓

URL만 입력하면, 나머지는 AI가 합니다

✓

AI가 서비스를 직접 탐방합니다. URL만 입력하면 화면을 하나씩 돌아봅니다

✓

화면별 맥락을 자동으로 수집합니다. UI, 스토리, 정책이 자동 생성됩니다

✓

서비스 전체 구조를 트리뷰로 시각화합니다. 한눈에 전체 구조를 파악할 수 있습니다

03
