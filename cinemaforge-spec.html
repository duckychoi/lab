핵심 개념 & 사용자 경험

🎬
3D 촬영장 시뮬레이션
영화 세트장처럼 3D 공간에 카메라들이 배치됨. 각 카메라 위치에 실제 촬영될 씬의 썸네일이 보임 (Rerun 스타일 frustum). 마우스로 자유롭게 워크스페이스를 탐색.
OrbitControls CameraFrustum R3F
📸
카메라 포즈 시각화
카메라마다 3D 공간에 절두체(frustum) 메시가 렌더링됨. 절두체 안쪽 면에 해당 샷의 이미지가 텍스처로 붙어 보임. Rerun / Nerfstudio 방식과 동일.
PinholeCamera PlaneGeometry Texture
〰️
카메라 궤적 실시간 표시
씬 순서에 따라 카메라 경로가 스플라인 곡선(CatmullRom)으로 3D 공간에 그려짐. 재생 시 카메라가 이 경로를 따라 이동. 궤적 직접 드래그로 수정 가능.
CatmullRomCurve3 TubeGeometry
▶️
재생 / 일시정지 / 편집
Play: 카메라가 궤적을 따라 이동하며 씬이 동영상으로 보임.
Pause: 현재 프레임에서 멈추고 편집 모드로 전환.
편집 내용은 3D 워크스페이스에 즉시 반영.
useFrame AnimationMixer
🃏
샷 카드 패널
오른쪽 패널에 씬별 카드가 세로로 나열됨. 각 카드에는 썸네일 + 프롬프트 입력창 + 참조 이미지 드래그 업로드. 카드 순서 드래그로 씬 순서 변경.
DnD Kit Card UI Prompt
🔄
2D ↔ 3D 양방향 동기화
카드에서 프롬프트 입력 → AI 생성 → 해당 카메라 위치 frustum 텍스처 업데이트.
3D에서 카메라 이동 → 카드 썸네일 갱신.
상태는 Zustand로 단일 관리.
Zustand WebSocket Sync
🖼️
에셋 드래그 & 드롭
이미지, 3D 모델(.glb/.obj), 영상 파일을 워크스페이스나 카드로 드래그해서 가져올 수 있음. 이미지는 3D 공간에 빌보드로 배치하거나 카드 참조 이미지로 등록.
useDropzone GLTFLoader
🎥
AI 씬 생성 & 편집
카드의 이미지 + 프롬프트를 AI에 전달 → FLUX로 이미지 생성 / Wan2.1로 영상 생성. 생성된 결과를 3D 씬에 즉시 반영. 인페인팅으로 부분 수정도 가능.
FLUX.1-dev Wan2.1-I2V IC-Light
